ZipPasswordRecall
==========
ZipPasswordRecall は忘れてしまったzipパスワードを思い出すためのアプリケーションです。  
  
ソフト詳細説明  
---------------------------------
ZipPasswordRecall は忘れてしまったzipパスワードを思い出すためのアプリケーションです。  
このアプリケーションは作者自身が利用するために実装しました。  
  
当初は Lhaplus の"ZIP パスワード検索"を利用しましたが、数日が経過しても完了しませんでした。  
(計算上、作者が利用するPCのスペックでは、10桁程度のパスワード文字数で数百年かかることが分かりました...。)  
そのため、覚えていた接頭辞を指定できれば(総当たりの桁数が少なくなることにより)すぐに完了するのでは？と思いつき、このアプリケーションを実装するに至りました。
